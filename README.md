# Installation
## Linux
```sh
> git clone https://github.com/CodeSteak/conio
> cd conio
> make
> sudo make install
```

`libconio.so` gets installed to `/usr/lib/conio/libconio.so`.
A link will be created at `/usr/lib/libconio.so`.
`conio.h` gets copied into `/usr/include/`.

For uninstall run:
```sh
> sudo make uninstall
```
